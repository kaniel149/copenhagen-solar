📸 שלב 1: סקר גג — לפני הכל

צילום, מדידות, הערכה מבנית

מה בודקים

📐 מדידות פיזיות

שטח גג, זווית שיפוע (tilt), כיוון (azimuth). אופטימלי בתאילנד: 10-15° שיפוע, דרום.

🏗️ מצב מבני

סוג גג (רעפים, מתכת, שטוח), גיל, עומס מותר. בתאילנד: בדיקת נזקי מונסון קודמים.

🌳 הצללות

עצים, מבנים סמוכים, אנטנות. שימוש ב-Solar Pathfinder או SunEye.

⚡ לוח חשמל

מיקום Main Panel, גודל Main Breaker, מרחק מהגג. חיבור חד/תלת פאזי.

📊 חשבונות חשמל

12 חודשים אחרונים. דפוס צריכה: פיק, בסיס, עונתיות.

📐 שלב 2: תכנון והנדסה

סימולציה, עיצוב חשמלי, חישובים מבניים

תהליך תכנון

1. סימולציית PVsyst — ייצור שנתי, Performance Ratio, הפסדי הצללה, טמפרטורה (חשוב בתאילנד — ממוצע 35°C).

2. String Configuration — חישוב כמות פאנלים בסטרינג, Voc בקור (בתאילנד min ~22°C), MPPT matching.

3. Inverter Sizing — DC/AC ratio (1.1-1.3 בתאילנד), MPPT ranges, transformer matching.

4. SLD — Single Line Diagram — סכמה חשמלית מלאה: DC isolators, AC breakers, SPD, earthing, meter.

5. Layout Drawing — מיקום פאנלים על הגג, מרווחים (wind gaps), מסלולי כבלים.

6. Structural Calc — עומס רוח (Koh Phangan zone), עומס עצמי, עומס גשם, עיגון.

תוכנות

PVsyst 8

סימולציית ייצור מדויקת. נתוני מזג אוויר TMY Koh Phangan. Output: דו"ח ייצור שנתי.

Helioscope

עיצוב מהיר online. פריסת פאנלים על תמונת drone. הצעת מחיר מיידית.

AutoCAD / SketchUp

שרטוט SLD, layout drawings, הגשה ל-PEA.

🏗️ שלב 3: קונסטרוקציה — שלד הרכבה

ימים 1-2 של עבודה בשטח

לפי סוג גג

גג רעפים (Tile Roof)

הוקים ייעודיים (tile hooks) מתחת לרעף. לא קודחים דרך הרעף! הוק → rail → clamp → panel.

נפוץ ברזורטים ווילות סגנון תאילנדי.

גג מתכת (Metal Roof)

קלאמפים על ה-standing seam או L-feet + bolts. איטום סביב חורים חובה (butyl tape + sealant).

נפוץ במבני תעשייה ומחסנים.

גג שטוח (Flat Roof)

שתי אפשרויות: Ballast (משקולות, בלי קדיחה) או Tilt frame (שלד בזווית 10-15°). ב-Koh Phangan: tilt frame עדיף — ניקוז גשם.

חומרים — Marine Grade

קריטי באי: אוויר מלוח = קורוזיה מואצת. רק אלומיניום 6005-T5 (rails) + נירוסטה 316 (brgs). ברגים מגולוונים = חלודה תוך שנה!

⚡ שלב 4: חשמל DC

שרשור פאנלים, כבלים, מפסקים

רצף עבודה

1. הנחת פאנלים על Rails (clamps מהודקים ל-35 N·m)

2. חיבור MC4 — שרשור פאנלים בסטרינג

3. ניתוב כבלי DC בתעלות (UV-resistant conduit)

4. התקנת DC Isolator ליד האינוורטר

5. הארקת שלד (PE conductor → earthing rod)

6. בדיקת Voc + Isc כל סטרינג (multimeter)

7. בדיקת בידוד (insulation resistance > 1MΩ)

זמן: 1-2 ימים לצוות של 4 (מערכת 30kWp)

מפרט כבלים

אזהרה: כבלי DC תחת מתח גם כשהמערכת "כבויה" (כל עוד יש אור). עבודה רק עם כפפות 1000V + כלים מבודדים. אין לעבוד ב-DC בגשם!

🔌 שלב 5: חשמל AC — חיבור לרשת

מהאינוורטר ועד הלוח הראשי

רצף עבודה

1. התקנת אינוורטר (Huawei SUN2000) — קיר מוצל, מאוורר

2. חיבור סטרינגים DC → MPPT inputs

3. AC Isolator — ליד האינוורטר

4. כבל AC → Main Distribution Board

5. Solar Breaker — dedicated, מתחת ל-Main Breaker

6. SPD Type I+II (הגנת ברקים — קריטי באי!)

7. CT Clamp / Smart Meter — לניטור ייצוא

8. חיבור תקשורת — WiFi / Ethernet → FusionSolar

חיבור פאזות

חד-פאזי (< 5kW)

וילות קטנות. L + N + PE. Breaker 32A.

תלת-פאזי (5-100kW)

רזורטים ועסקים. 3L + N + PE. Balance בין פאזות. רוב ההתקנות שלנו.

Huawei SUN2000 יתרונות:
▸ 2-6 MPPT trackers (גמישות בשרשור)
▸ Built-in WiFi + 4G dongle
▸ FusionSolar monitoring חינם
▸ IP65 — מתאים לחוץ
▸ 10 שנות אחריות (15 בהרחבה)

✅ שלב 6: חיבור PEA + Net Metering

הרגולטור — Provincial Electricity Authority

תהליך PEA (מפושט מ-11/2025)

1. הגשת בקשה — טופס + SLD + מפרט טכני

2. PEA בודק מסמכים (3-5 ימי עבודה)

3. בדיקה בשטח — מפקח PEA מגיע לאתר

4. אישור → החלפת מונה לדו-כיווני

5. Commissioning Test — בדיקת Anti-Islanding

6. חיבור לרשת — LIVE!

חדש (2025): אין צורך בהיתר בנייה, אין רישיון מפעל. רק תיאום PEA. זמן: 7-14 יום מהגשה עד חיבור.

Net Metering בתאילנד

איך זה עובד

עודף ייצור → נזרק לרשת → מונה חוזר אחורה. יחס 1:1 — כל kWh שמיוצא = kWh חינם בלילה.

הגבלות

מערכת ≤ גודל החיבור הקיים. ייצוא לא עולה על ייבוא שנתי. עודף סוף שנה → לא מפוצה.

🔍 שלב 7: בקרת איכות + מסירה

בדיקות סיום

IV Curve Testing

בדיקת ביצועי כל סטרינג מול מפרט יצרן. זיהוי פאנלים פגומים.

Thermal Imaging

מצלמת FLIR — זיהוי hot spots, חיבורים רופפים, תאים פגומים. חובה לפני מסירה.

Performance Ratio

השוואת ייצור בפועל מול סימולציה. PR צפוי: 75-82% (תאילנד — טמפרטורה גבוהה).

Punch List

רשימת ליקויים → תיקון → אישור לקוח → מסירה.

⛑️ בטיחות — סביבה טרופית

אתגרים ייחודיים לאי

חום קיצוני

35-42°C על הגג. הפסקות כל שעה. מים + אלקטרוליטים. עבודה 7:00-11:00 + 14:00-17:00 בלבד.

עונת מונסון (אוק-דצמ)

גשמים כבדים. אין עבודת DC בגשם. תכנון עבודות לעונה יבשה (ינואר-ספט).

ברקים

קורוזיה מואצת. בדיקת חיבורים חשמליים כל 6 חודשים. רק SS316 + aluminum.
