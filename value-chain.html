📥 שלב 1: Lead Capture — לכידת ליד

סטטוס: קיים ב-CRM | צריך: webhooks + auto-assign

מה קורה

  • ליד נכנס מ-WhatsApp / LINE / Facebook / אתר / Roof Scanner
  • נרשם אוטומטית עם מקור + תאריך
  • שיוך אוטומטי לאיש מכירות
  • התראה תוך 2 שעות אם לא נענה

סטטוסים

New Contacted Survey Scheduled Qualified Lost

מה לבנות

  • WhatsApp Business webhook → auto-create lead
  • LINE Official webhook → auto-create lead
  • Landing page form → API → lead
  • Roof Scanner → "Request Quote" → lead
  • Auto-assign round-robin
  • SLA timer: 2h response alert

📸 שלב 2: Site Survey — סקר אתר

סטטוס: לבנות | עדיפות: Phase 1

צ'קליסט סקר

  • צילום גג מלמעלה (דרון)
  • צילום גג מקרוב (סוג חומר)
  • צילום ארון חשמל — פנים + חוץ
  • צילום מונה PEA — מספר + קריאה
  • צילום סביבה — הצללות, עצים
  • מדידות: אורך × רוחב × זווית גג
  • כיוון גג (compass)
  • מרחק: גג → ארון → מונה
  • סוג גג: בטון / פח / רעפים / עץ
  • מצב גג: טוב / בינוני / דורש תיקון
  • גישה: סולם / מדרגות / מנוף

מסכי CRM

  • טופס סקר מובנה (mobile-friendly)
  • Upload תמונות עם סיווג אוטומטי
  • מפה עם GPS pin
  • גלריית תמונות per lead
  • צ'קליסט — חייב 100% לפני שלב הבא

שדות מפתח

roof_type, roof_area_m2, roof_angle panel_distance_m, meter_number shading_hours, roof_condition lat, lng, photos[]

⚡ שלב 3: Electricity Analysis — ניתוח חשמל

סטטוס: לבנות | עדיפות: Phase 1

מה אוספים

  • 3-12 חשבונות PEA (חודש, kWh, THB)
  • תמונת חשבון (OCR עתידי)
  • סוג תעריף: residential / commercial / TOU
  • Demand charge + FT surcharge

מה מחשבים (אוטומטי)

  • צריכה ממוצעת חודשית (kWh)
  • עלות ממוצעת per kWh
  • גרף צריכה חודשי + מגמות
  • גודל מערכת מומלץ (kWp)
  • חיסכון שנתי צפוי (THB)
  • CO₂ reduction

🎨 שלב 4: System Design — עיצוב מערכת

סטטוס: לבנות | עדיפות: Phase 2

Panel Layout Designer

  • פריסת פאנלים על תמונת גג / מפה
  • Drag & drop פאנלים
  • חישוב אוטומטי: כמות + הספק
  • אזורי הצללה מסומנים
  • מרחק שורות (row spacing)
  • תמונת layout לשליחה ללקוח

בחירת ציוד

  • Panel: LONGi 580W / 550W / 400W
  • Inverter: Huawei 5-20KTL
  • Battery: LUNA2000 5-15 kWh
  • Mounting: roof / ground / ballast

חישובי ביצועים

  • Total kWp, annual kWh
  • Self-consumption ratio
  • Grid export ratio
  • Performance ratio

📐 שלב 5: SLD — תוכנית חד-קווית

סטטוס: לבנות | עדיפות: Phase 2

SLD Generator

  • אוטומטי מעיצוב המערכת
  • תבנית: residential / commercial / hybrid
  • DC cable sizing
  • AC cable sizing
  • Breaker ratings
  • SPD + Grounding
  • Export: SVG + PDF

מעקב PEA

  • תאריך שליחה
  • מספר בקשה
  • מועד בדיקה
  • סטטוס: Preparing → Submitted → Approved

💰 שלב 6: Proposal — הצעת מחיר

סטטוס: קיים (proposal.html) — להרחיב ולשלב ב-CRM

Proposal Builder

  • Auto-fill מעיצוב + ניתוח חשמל
  • מודל EPC: מחיר, תשלומים, payback
  • מודל PPA: הנחה, חיסכון, חוזה
  • השוואה EPC vs PPA side-by-side
  • תמונת layout על הגג
  • Branding: TM Energy
  • שליחה ב-WhatsApp / Email

מודלים כלכליים

  • EPC: NPV, IRR, payback years
  • PPA: monthly rate, escalation, IRR
  • Sensitivity: best / base / worst
  • חתימה דיגיטלית

📦 שלב 8: Procurement — רכש ומלאי

סטטוס: לבנות | עדיפות: Phase 3

ניהול מלאי

  • פאנלים, ממירים, סוללות, מרכבים, כבלים
  • במחסן / בדרך / משויך / זמין
  • התראת מלאי מינימלי
  • שיוך ציוד לפרויקט (pull from stock)
  • Serial number tracking

הזמנות רכש (PO)

  • ספק, פריטים, מחיר, delivery date
  • סטטוס: Draft → Sent → Shipped → Received
  • מעקב משלוח (מעבורת לאי)
  • היסטוריית מחירים per item

🔨 שלב 9: Installation — התקנה

סטטוס: לבנות | עדיפות: Phase 3

צ'קליסט התקנה

  • הכנת גג (ניקוי, חיזוק)
  • התקנת מרכבים
  • הנחת פאנלים
  • חיווט DC (strings)
  • התקנת ממיר
  • חיווט AC
  • סוללה (אם יש)
  • Grounding + SPD
  • בדיקות DC: Voc, Isc
  • בדיקות AC: voltage, frequency
  • בדיקת בידוד

ניהול צוות

  • Team lead + crew assignment
  • Gantt פשוט — שלבי התקנה
  • תמונות per שלב
  • דיווח בעיות + פתרון
  • שעות עבודה + הוצאות

🔌 שלב 10: PEA Connection — חיבור לרשת

סטטוס: לבנות | עדיפות: Phase 4

מעקב PEA

  • הגשת בקשה (30 יום הודעה)
  • מעקב מספר בקשה
  • תיאום בדיקה
  • תוצאת בדיקה: pass / fail / conditional
  • התקנת מונה דו-כיווני
  • אישור net metering

מסמכים נדרשים

  • SLD (מהשלב 5)
  • צילום תעודת זהות
  • ספר בית (house book)
  • טופס PEA
  • תמונות המערכת המותקנת

✅ שלב 11: Commissioning — הפעלה ומסירה

סטטוס: לבנות | עדיפות: Phase 4

צ'קליסט הפעלה

  • בדיקת ייצור (kW actual vs expected)
  • הגדרת monitoring (FusionSolar / iSolarCloud)
  • הדרכת לקוח — app, מה לעשות בהפסקה
  • מסירת מסמכים: אחריות, SLD, manual
  • חתימת פרוטוקול מסירה (דיגיטלי)
  • תיעוד תמונות סופי

ביצועים יום 1

  • kWh produced
  • Peak kW
  • Performance ratio %
  • Monitoring configured
  • Customer trained

Dashboard ניטור

  • ביצועים vs expected

  • kWh, חיסכון, CO₂, ביצועים
  • שליחה ב-WhatsApp / Email
  • חיוב O&M חודשי (invoice)
