לפי סוג גג

רעפים (Tile)

Hook מתחת לרעף → rail → clamp. לא קודחים דרך הרעף. נפוץ ברזורטים.

מתכת (Standing Seam)

Clamp ישירות על seam. בלי חורים = בלי נזילות. מחסנים ותעשייה.

מתכת (Corrugated)

L-foot + bolt דרך הגג. חובה: butyl tape + sealant. איטום קריטי.

שטוח (Flat)

Tilt frame 10-15°. בלסט או עיגון. ניקוז גשם חשוב בטרופי.

Wind Load: Zone 2 = עד 130 km/h. עיגון כל 1.2m. Rail spacing ≤ 1.4m.

📐 תהליך עיצוב מערכת

6 שלבים

1. Survey Data → תמונות drone, מדידות, חשבונות חשמל, סוג גג

2. PVsyst Simulation → ייצור שנתי, PR, הפסדים (temp, shading, soiling). TMY data: Koh Phangan (9.7°N, 100.0°E)

3. String Design → כמות פאנלים/string, Voc range vs MPPT window, temperature correction (min 22°C, max 45°C cell)

4. SLD (Single Line) → סכמה חשמלית: DC strings → isolators → inverter → AC isolator → SPD → breaker → grid

5. Layout Drawing → פריסת פאנלים על הגג. מרווחים: wind gaps, fire access, maintenance access

6. BOM + Quote → Bill of Materials מלא → הצעת מחיר → CRM → PDF ללקוח
